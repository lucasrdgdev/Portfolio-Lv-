// 01 — sobre

Quem escreveu este código

Sou Lucas Rodrigues Velozo, Estagiário de Desenvolvimento na Avantia, onde atuo com JavaScript, integrações de APIs REST e automação de processos na plataforma Sydle One. Em paralelo, curso Análise e Desenvolvimento de Sistemas na UNIT e participo do programa AWS re/Start, aprofundando fundamentos de Cloud e infraestrutura.

Sou Residente Tecnológico no Porto Digital e fundador da DEV LINK, comunidade voltada a conectar pessoas desenvolvedoras e fortalecer o ecossistema tech local. Este portfólio foi construído inteiramente com HTML, CSS e JavaScript nativos — sem frameworks — como exercício prático dos fundamentos da web.

  • Avantia Estágio em Desenvolvimento — JavaScript, APIs REST, Sydle One
  • UNIT Graduando em Análise e Desenvolvimento de Sistemas
  • AWS re/Start Formação em fundamentos de Cloud
